# Internet Terminology Project

This project is a multi-page website that elaborates on fundamental Internet Terminology.

## Project Structure
- `index.html`: The home page of the website.
- `style.css`: The stylesheet ensuring consistent design across all pages.
- `images/`: A directory containing SVG illustrations for each terminology page.
- Individual HTML files for each term (e.g., `www.html`, `dns.html`, etc.).

## Key Features
- **Comprehensive Coverage**: 16 individual pages for various internet terms.
- **Interlinking**: A navigation bar on every page allows seamless navigation through the entire site.
- **Visual Aids**: Every page includes a relevant SVG image to enhance the explanation.
- **Design Consistency**: A fixed navigation bar and shared CSS ensure a professional user experience.
